To use this awesome application
Select a few stocks from the drop down list (Stocks), the Stocks are the ticker symbols
Select a Date Range that will control the time period the stock data is extracted and analysed.
The End Date must be greater than the start Date
Then click on Chart!
Tha application will produce two charts,
The top chart shows the price series of the selected stocks and is an interactive chart that the user can zoom in on
The bottom chart will plot an efficient frontier of stocks, this is the best stock allocation you can get given the selected stocks.
It will also plot the optimal portfolio result as determined by the Sharpe Ratio.
